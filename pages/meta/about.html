Our Mission

This wiki project aims to create a collaborative knowledge base that is accessible, editable, and useful for everyone. We believe in the power of collective intelligence and the importance of sharing information freely.

History

The Wiki Project was created in March 2025 as a lightweight alternative to traditional wiki systems. It was designed specifically to work with static hosting services like GitHub Pages, making it easy for individuals and small teams to create and maintain their own wiki without complex server setups.

How It Works

Unlike traditional wiki systems that require server-side databases, this wiki uses client-side storage (localStorage) to save changes made by users in their own browsers. This means that:

  • Changes are saved in the user's browser
  • Edits are personal to each user and device
  • No account creation is necessary
  • No server-side processing is required

For collaborative editing across multiple users, changes need to be manually shared and merged. This is typically done by having contributors submit their changes through GitHub pull requests, which can be reviewed and merged by maintainers.

Technology

The wiki uses a combination of technologies:

  • HTML/CSS/JavaScript: Core foundation of the wiki
  • Marked.js: For rendering Markdown as HTML
  • DOMPurify: For sanitizing user-submitted HTML
  • localStorage API: For client-side storage of user edits

Contributing

Anyone can contribute to this wiki by editing pages directly in the browser. For those familiar with GitHub, you can also fork the repository, make changes, and submit a pull request. Please review our guidelines before contributing.

Future Development

We're planning to add several features in the future:

  • Better handling of edit conflicts
  • Page history and versioning
  • Automatic link creation between pages
  • Improved media handling capabilities
  • Search functionality
